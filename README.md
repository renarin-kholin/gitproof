# GitProof

Developer Rating System that evaluates GitHub activity, analyzes code quality, and generates detailed performance metrics using AI.

## Features

- GitHub profile analysis
- Code quality metrics
- AI-powered insights
- Developer leaderboard
- Compare developers side-by-side

## Getting Started

To run this application:

```bash
npm install
npm run dev
```

## Building For Production

```bash
npm run build
```

## Testing

```bash
npm run test
```

## Linting & Formatting

```bash
npm run lint
npm run format
npm run check
```
